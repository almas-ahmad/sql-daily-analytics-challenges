# SQL Daily Analytics Challenges

This repository contains daily SQL practice focused on real-world business problems.  
Each day includes a problem statement, sample dataset, and SQL solution.

## Objective
To build strong SQL skills required for Data Analyst roles by solving practical analytics scenarios.

## Topics Covered
- Data Cleaning & Validation
- Aggregations & Grouping
- Joins & Subqueries
- Window Functions
- Cohort & Retention Analysis
- Churn Detection
- Fraud & Anomaly Detection
- Revenue & Trend Analysis
- Domain-based Analytics (Fleet, Fuel, Transactions)

## Tools Used
- SQL (PostgreSQL / Athena / MySQL compatible)
- CSV datasets

## Goal
Maintain consistent daily practice and demonstrate hands-on SQL problem-solving for real business use cases.
